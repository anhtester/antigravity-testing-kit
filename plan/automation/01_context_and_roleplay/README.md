# Bước 1: Khởi tạo ngữ cảnh (Context Initialization)

**Mục đích:**
Định hình cho AI vai trò của một Automation Engineer thực thụ. Bước này rất quan trọng để AI biết bạn đang dùng ngôn ngữ lập trình nào, framework nào (Playwright, Selenium, Appium...) để sinh code với cú pháp chính xác.

**Cách sử dụng:**
1. Mở file `prompt.txt`.
2. Thay thế thông tin framework, ngôn ngữ và bối cảnh dự án vào trong `[...]`.
3. Gửi cho AI và chờ AI xác nhận hiểu rõ bối cảnh rồi mới sang Bước 2.
