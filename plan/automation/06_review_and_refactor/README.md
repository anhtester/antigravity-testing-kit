# Bước 6: Kiểm duyệt & Tối ưu (Review & Refactoring)

**Mục đích:**
Sau khi mã test tự động chạy thành công, chúng ta cần "dọn dẹp" mã nguồn để chuẩn bị đưa vào dự án chính thức (merge vào nhánh master). Tránh để lại mã rác làm bẩn project.

**Cách sử dụng:**
1. Dán toàn bộ source code (Page + Test + Utils) lại cho AI 1 lần nữa sau khi bạn chạy test Pass cục bộ.
2. AI giúp review theo tiêu chuẩn Clean Code và CI-ready.
