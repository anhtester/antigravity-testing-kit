# Bước 5: Sinh mã Automation (Script Generation)

**Mục đích:**
Bước cốt lõi để AI output ra đoạn code Test thực thi dựa trên các Page Objects và Test Data đã chuẩn bị. Đảm bảo có các lệnh kiểm tra (Assertions) đúng vị trí.

**Cách sử dụng:**
1. Tổng hợp lại các phần trước và yêu cầu AI generate ra Test Class.
2. Nhấn mạnh việc xài Assertions và Wait chuẩn.
