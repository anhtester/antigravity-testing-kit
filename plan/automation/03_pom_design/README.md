# Bước 3: Thiết kế cấu trúc POM (POM Design)

**Mục đích:**
Sau khi AI đã tự lấy chính xác danh sách Elements từ UI bằng trình duyệt của máy ở Bước 2, hướng dẫn AI định hình các Class theo chuẩn kiến trúc Page Object Model (POM). Bước này đảm bảo source phân bổ đúng file, tên hàm có ý nghĩa rõ ràng trước khi cài logic cụ thể.

**Cách sử dụng:**
1. Cầm bản danh sách Locator mà AI báo cáo lại từ Bước 2 (kiểm định sơ qua).
2. Nối tiếp ra lệnh bằng nội dung trong `prompt.txt` này để AI chép dữ liệu đó thành kết cấu file/mô hình Class.
