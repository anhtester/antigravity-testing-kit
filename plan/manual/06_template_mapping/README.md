# Bước 6: Chuẩn hóa Format (Template Mapping)

**Mục đích:**
Đóng gói toàn bộ các Test Cases đã được AI tạo đoạn Text/Markdown sang định dạng Bảng (Table Format) dễ đọc nhất, chuẩn bị cho việc sao chép ra Excel, Google Sheets hoặc import thẳng lên các Test Management Systems.

**Cách sử dụng:**
1. Chạy prompt trong file `prompt.txt` này để AI vẽ bảng.
2. Copy kết quả bảng và paste trực tiếp vào công cụ quản lý test của dự án bạn.
