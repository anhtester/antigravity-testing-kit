# Bước 5: Sinh Test Case chi tiết (RBT & Test Case Generation)

**Mục đích:**
Thiết kế Test Case chi tiết dựa trên chiến lược kiểm thử rủi ro (Risk-Based Testing). Rủi ro cao thì test kĩ, rủi ro thấp thì test cơ bản.

**Cách sử dụng:**
1. Copy nội dung `prompt.txt` và điền định hướng test data bạn mong muốn, sau đó gửi cho AI.
2. Cảnh báo: Nếu số lượng kịch bản ở Bước 4 quá nhiều, AI có thể sinh không hết. Lời khuyên là hãy yêu cầu AI sinh Test Case cho từng Module một (Ví dụ: Hãy sinh TC cho Module 1 và 2 trước tiên).
