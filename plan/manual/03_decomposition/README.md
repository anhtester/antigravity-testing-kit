# Bước 3: Phân rã hệ thống (Decomposition Strategy)

**Mục đích:**
Hướng dẫn AI chia nhỏ một hệ thống/tính năng phức tạp thành nhiều Module hoặc Feature nhỏ hơn. Tránh tình trạng AI bị "ngợp" thông tin dẫn đến viết test case thiếu sót.

**Cách sử dụng:**
1. Chắc chắn rằng các câu hỏi ở Bước 2 đã được bạn trả lời và làm rõ cho AI.
2. Gửi file `prompt.txt` cho AI để phân rã cấu trúc tính năng thành các phần dễ quản lý.
